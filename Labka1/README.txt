НАЗНАЧЕНИЕ ПРОГРАММЫ:
1. Генерация всех латинских квадратов для n=4,5,6
2. Проверка каждой таблицы на свойства группы
3. Классификация групп по изоморфизму
4. Генерация CSV отчетов по ТЗ

ОСНОВНЫЕ ОПТИМИЗАЦИИ:

1. ЛАТИНСКИЕ КВАДРАТЫ (ключевая оптимизация):
   - Всего таблиц для n=6: 6^36 ≈ 10^28 (невозможно)
   - Латинских квадратов: ~800 миллионов (реально)
   - Сокращение в 10^20 раз!

2. ПАРАЛЛЕЛИЗАЦИЯ:
   - Генерация: Parallel.ForEach по первым строкам
   - Проверка групп: Parallel.ForEach по таблицам
   - Классификация: предварительный параллельный расчет канонических форм

3. АЛГОРИТМИЧЕСКИЕ ОПТИМИЗАЦИИ:
   - Раннее отсечение при проверке групп
   - Канонические формы для быстрого сравнения
   - Backtracking с эвристиками для генерации

ТРЕБОВАНИЯ К СИСТЕМЕ:
- .NET 8.0 Runtime
- 8+ ГБ оперативной памяти (для n=6)
- 20+ ГБ свободного места на диске
- Многоядерный процессор (чем больше ядер, тем быстрее)

ОЖИДАЕМОЕ ВРЕМЯ ВЫПОЛНЕНИЯ:
- n=4: 1-2 минуты
- n=5: 10-20 минут
- n=6: 2-6 часов (в зависимости от CPU)

ЗАПУСК ПРОГРАММЫ:
1. Скомпилировать: dotnet build --configuration Release
2. Запустить: CayleyTableAnalyzer.exe
3. Результаты появятся в папке CayleyResults/
